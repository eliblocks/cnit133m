Service Workers

Service Workers is a new web technology that has a lot of people excited about the possiblitiy of spamming website visitors with promotional push notifications after their users have blocked app notifications on their phones. This technology also has the lesser significance of enabling better offline experiences, replacing functionalites offered by the deprecated AppCache API and adding many new capabilities.

One of the Highlights are the caching capabilities. Service Workers allow granular control over which assets wil be cached and enable chaches to be updated even when the user is not on the site.

Service workers are very different from the traditonal web request and response cycle. Instead it is asyncronous and event driven. Requests can be triggered to urls of the developers choice at any time, without user input.
